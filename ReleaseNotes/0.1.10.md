### New
- Display Mode for macOS and Linux VMs: choose Fixed Size or Fit to Window. When Fit to Window is selected, the guest tracks the window and its last size is remembered and restored on the next launch.
- More memory sizes to choose from: 12, 24, 48, 96, 192, and 256 GB.

### Improved
- Memory choices leave headroom for the host instead of offering the machine's full RAM, and start at 8 GB for macOS guests.
- The Edit sheet is now "Configure," with settings split into Resources and Display.
