### New
- Retina display mode for macOS guests, matching the host's pixel density.
- Runner windows now close automatically when a VM powers off.

### Improved
- The sidebar shows each VM's run state at a glance, with a tinted running icon.
- Keyboard capture releases the moment a guest stops, so the ⌃ indicator resets.

### Fixed
- Window switching no longer strands a hidden runner window offscreen.
