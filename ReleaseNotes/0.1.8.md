### New
- Retina display option for macOS VMs — turn on Retina when creating or editing a VM to render the guest at full HiDPI density.
- A keyboard-capture indicator in the VM window toolbar shows where input is going: click inside the window to send keys to the guest, click the title bar to hand control back to the host. Full screen captures automatically, and the indicator clears when the guest stops.

### Improved
- The sidebar shows each VM's run state and tints a running VM's icon green, so you can see what's live at a glance.
- Stopping a VM (a guest shutdown or the toolbar Stop) now leaves its window open on a Powered Off screen so you can restart it in place; closing the window with the red button powers the VM off and closes it.
